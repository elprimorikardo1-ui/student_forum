# Student Forum (Static)

Минимальный проект студенческого форума в виде статического SPA на чистом JavaScript.
Работает полностью в браузере. Данные сохраняются в `localStorage`.

## Функции
- Создание тредов
- Просмотр списка тредов
- Присоединение к треду
- Добавление комментариев к треду
- Регистрация через логин + пароль
- Выбор аватарки при регистрации
- Сохранение данных в localStorage
- Работа офлайн / без сервера

## Запуск локально
1. Откройте `index.html` в браузере.
2. Или, если хотите, через простой сервер (рекомендуется):

```bash
python -m http.server 8000
```

3. Откройте `http://localhost:8000`.

### Пользовательская авторизация (без почты)
1. Введите логин и пароль.
2. Кнопка `Войти` для существующего аккаунта.
3. Кнопка `Регистрация` создаёт нового пользователя (логин уникален).
4. После входа создавайте треды и комментируйте.

> Треды общие для всех пользователей в одном браузере. Если открыть сайт в другой вкладке, новые треды появятся автоматически.

## GitHub и Netlify
1. Установите `git` и откройте папку проекта.
2. Выполните:

```bash
git init
git add .
git commit -m "Initial student forum with auth and avatars"
```
3. Создайте репозиторий на GitHub.
4. Добавьте удалённый репозиторий и запушьте:

```bash
git remote add origin https://github.com/<ваш-пользователь>/<ваш-репозиторий>.git
git branch -M main
git push -u origin main
```

## Деплой на Netlify
1. Зарегистрируйтесь на https://netlify.com.
2. New site -> Import from Git -> выберите ваш репозиторий.
3. Build command: оставьте пустым.
4. Publish directory: `.`
5. Деплой начнётся автоматически.

> `netlify.toml` уже настроен для SPA.
